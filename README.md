# rh-template-gis

GIS + MIS 工程模板

- 在线 demo: https://gis.giscafer.com

## 技术栈

- Umi 4.x
- React18 + TypeScript + antd4.x
- qiankun 微前端

## 使用说明

node 版本推荐: `v16.17.0`

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn dev
```

Build

```bash
$ yarn build
```
